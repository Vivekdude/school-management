# Kalangan Dance Institute

This is the website for **Kalangan Dance Institute**, built with [Next.js](https://nextjs.org) and styled using Tailwind CSS. It provides information about classes, instructors and contact details for the institute.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

The landing page is located at `src/app/page.js` and includes a hero section, class listings and contact information. Styling is handled with Tailwind CSS and the [Geist](https://vercel.com/font) typeface.
